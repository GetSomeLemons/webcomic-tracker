Webcomic Tracker — Help

Quick guide to getting everything working.

Saving a comic page

The extension works on any web page — no setup required.

Option 1 — Track button (always works) Open the popup (click the extension icon) and press Track. The current page is saved with its title and URL. Chapter number is detected automatically on asurascans.com.
Option 2 — Keyboard shortcut Alt+S Works on any tab. If pressing Alt+S does nothing, follow the steps below.

Fixing Alt+S

  1. Go to edge://extensions/shortcuts (Edge) or chrome://extensions/shortcuts (Chrome).
  2. Find Webcomic Tracker and set the shortcut for "Save or update this comic page".
  3. If the page was open before the extension was installed, refresh the tab.

Keyboard element binding (⌨)

Bind any button on any page to a keyboard key — for example, bind → to "Next chapter".

  1. Open the popup and click ⌨ (top right).
  2. Click 🎯 Pick element on page — the popup closes.
  3. The page dims. Hover over the element you want to bind (e.g. "Next ›"). It lights up in blue.
  4. Click it.
  5. Press the key you want to bind (e.g. ] or ArrowRight).
  6. A confirmation toast appears. The binding is now active on this site.

Manage or delete bindings from the ⌨ view in the popup.

Cloud sync optional

Your comic list syncs to a private GitHub Gist. Set it up once and your data follows you across devices.

Step 1 — Create a GitHub Personal Access Token

  1. Go to github.com/settings/tokens.
  2. Click Generate new token → Fine-grained token.
  3. Set expiry as desired. Under Account permissions, set Gists → Read and write.
  4. Click Generate token and copy the token (starts with github_pat_…).

Step 2 — Connect in Settings

  1. Open the popup → ⚙ → Settings.
  2. Paste the token into Personal Access Token.
  3. Click Connect. A Gist is created (or an existing one is found).
  4. Repeat on other devices/profiles with the same token.

Dark mode

Click 🌙 in the popup to toggle dark mode on the current tab. To enable it globally on all pages, go to Settings → Enable dark mode on all pages globally.

Uses a CSS filter (invert + hue-rotate). Comic images are double-inverted so they display normally. Not every site will look perfect — it covers ~95% of cases.
