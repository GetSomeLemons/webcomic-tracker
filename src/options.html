Keyboard Shortcut
Default shortcut: Alt+S — save current comic page.
If it doesn't work, configure it at edge://extensions/shortcuts (or chrome://extensions/shortcuts).

The page must be reloaded if the extension was installed after opening it. Also works without shortcut — use the Track button in the popup.
